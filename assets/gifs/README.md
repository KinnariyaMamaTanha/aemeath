# gif 的用途

除了 [seal.gif](./seal.gif) 之外，所有人物都用于显示人物。基于鼠标和当前人物的位置关系：

1. 当鼠标和人物相距较远时，显示 [move.gif](./move.gif)。当鼠标位置在人物右侧时，使用原始的 gif；当鼠标位置在人物左侧时，将 gif 左右翻转显示（这是因为 move.gif 本身是朝向右边的，翻转过后才能正确显示其移动）。且人物向鼠标位置进行速度恰当的移动（移动速度是用户可以调整的）
2. 当鼠标和人物相距较近时
   1. 首先人物在鼠标附近进行随机游荡一段时间t0，此时使用 move.gif，同时需要同第 1 点中一样注意左右翻转关系
   2. 然后人物在鼠标附近停下，并随机使用 idle*.gif 中的一个播放随机长的时间（但是有一个最短的播放时间），播放完当前 gif 后就随机切换为另一个 idle*.gif；所有 gif 被选择的概率相同
      1. 当鼠标长时间（超过时间t1）没有移动和没有操作时，播放 [idle2.gif](./idle2.gif) 的概率随时间线性提升，直到 100%；其余 idle*.gif 出现的概率保持相同
      2. 当鼠标更长时间（超过时间t2）没有移动和没有操作时，在屏幕上随机位置出现一只海豹（即 [seal.gif](./seal.gif)），人物不再跟随鼠标，而是跟随海豹并在海豹周围随机移动，以体现人物和海豹在进行互动
   3. 当鼠标进行选择、拖拽操作时，播放 [drag.gif](./drag.gif)
